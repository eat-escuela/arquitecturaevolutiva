Para hacer uso de este código siga los siguientes pasos.

1. Descargue un IDE de desarrollo de Java como Eclipse, Visual Studio Code, InteliJ, o el que prefiera.
2. Importe el código como proyecto de maven.
3. Para ejecutar las pruebas, ejecute el comando Run as-> Junit Test, en la clase src/test/java/com/eat/cuentas/FeatureRunner
4. Para ejecutar la aplicación, ejecute el comando Run as-> Java Application, en la clase src/main/java/com/eat/Application
